Facilities

  • shin-compute-000 is configured with 48-core CPUs, two NVIDIA GV100 GPUs, and 384 GB of RAM. This server is dedicated to prototyping code for high-performance computing clusters, aiming to solve a range of numerical problems, including large-scale simulation, optimization problems, inverse problems, and machine learning.
  • shin-compute-001 is configured with an AMD Radeon VII GPU. This server is dedicated to prototyping code for high-performance computing clusters with AMD GPUs, such as Frontier.
